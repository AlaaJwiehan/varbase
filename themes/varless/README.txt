WELCOME

1) please install latest npm.

   https://rtcamp.com/tutorials/nodejs/node-js-npm-install-ubuntu/

2) run the following command 

   sudo npm install -g grunt-cli

3) go to the config folder, and run the following command

   npm install

4) change the project name inside package.json

5) run the following command to compile css & js files.

   grunt dist
   
   OR

   grunt

6) run the following command to keep watching and compiling the project in realtime development
  
   grunt watch

NOTE : please do not commit the following folder (config/node_modules) this is added to the .gitignore just make sure you don't add it or remove it from the .gitignore.
